# Autonomous Vehicle Object Detection

## Project Description

This project detects objects such as cars, persons, buses, trucks and motorcycles using the YOLOv8 object detection model. It works on images, videos and live webcam input.

## Features

- Image Object Detection
- Video Object Detection
- Live Webcam Detection
- Object Counter
- FPS Display

## Technologies Used

- Python
- OpenCV
- YOLOv8
- Ultralytics

```
Autonomous-Vehicle-Object-Detection

```

## How to Run

1. Create a virtual environment.
2. Install the required libraries.

```
pip install -r requirements.txt
```

3. Run the project.

```
python video_detection.py
```
